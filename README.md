# GitReview

AI-powered tool that analyzes your GitHub profile from a recruiter's perspective.
Get an instant audit of your public repositories — overall score, breakdown by pillars
(README quality, project diversity, commit consistency, profile presentation),
top strengths, critical issues, and actionable recommendations per repo.

Built for developers in career transition who want honest feedback before sending their GitHub to a recruiter.
